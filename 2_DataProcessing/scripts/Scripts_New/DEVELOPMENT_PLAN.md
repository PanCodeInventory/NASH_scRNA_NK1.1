# 阶段脚本开发详细计划

## 阶段1：数据导入与预处理 (`01_data_import.R`)

### 核心功能
1. **读取10x数据**：安全读取四个时间点的10x数据
2. **创建Seurat对象**：为每个样本创建带时间点元数据的Seurat对象
3. **SCTransform归一化**：使用SCTransform进行数据归一化
4. **数据整合**：使用锚点法整合多个样本
5. **基础降维**：运行PCA和UMAP
6. **初始可视化**：生成ElbowPlot和基础UMAP图

### 输入
- 四个时间点的10x数据目录
- 配置文件参数（nfeatures、normalization_method等）

### 输出
- `nk.integrated.imported.rds` - 整合后的Seurat对象
- 基础可视化图件
- 数据质量报告

### 参考脚本
- 主要参考：`generate_umap_nk.R`
- 需要修复的硬编码路径问题

---

## 阶段2：基础质量控制 (`02_basic_qc.R`)

### 核心功能
1. **双胞检测**：使用scDblFinder检测双胞
2. **双胞去除**：移除标记为双胞的细胞
3. **质量控制报告**：生成双胞检测结果统计

### 输入
- 阶段1输出的Seurat对象
- 配置的doublet相关参数

### 输出
- `nk.integrated.qc_filtered.rds` - 去除双胞后的Seurat对象
- 双胞检测结果图件
- QC报告

### 参考脚本
- 主要参考：`remove_doublets_and_contaminants.R`
- 简化版本，专注于双胞检测

---

## 阶段3.1：细胞标注 (`03_cell_annotation.R`)

### 核心功能
1. **SingleR注释**：使用参考数据集进行细胞类型注释
2. **注释验证**：检查注释质量和分布
3. **细胞类型统计**：生成各细胞类型的数量统计

### 输入
- 阶段2输出的Seurat对象
- 配置的SingleR相关参数

### 输出
- `nk.integrated.annotated.rds` - 带注释的Seurat对象
- 细胞类型分布图
- 注释质量报告

### 参考脚本
- 主要参考：`singleR_annotation_fix.R`
- 需要修复路径问题

---

## 阶段3.2：特定细胞类型去除 (`04_cell_filtering.R`)

### 核心功能
1. **细胞类型过滤**：基于注释结果去除特定细胞类型
2. **规则配置**：支持可配置的保留/去除规则
3. **过滤验证**：生成过滤前后的对比统计

### 输入
- 阶段3.1输出的Seurat对象
- 配置的细胞类型过滤规则

### 输出
- `nk.integrated.filtered.rds` - 过滤后的Seurat对象
- 细胞类型组成变化图
- 过滤效果报告

### 参考脚本
- 主要参考：`remove_NKT_cells.R`
- 通用化为支持多种细胞类型过滤

---

## 阶段4：参数优化 (`05_parameter_tuning.R`)

### 核心功能
1. **网格搜索**：dims×resolution参数网格搜索
2. **多指标评估**：计算簇数、轮廓系数等指标
3. **候选选择**：基于多个指标选择最佳参数组合
4. **热力图可视化**：生成参数评估热力图

### 输入
- 阶段3.2输出的Seurat对象
- 配置的参数调优范围

### 输出
- 最佳参数组合
- 参数评估热力图
- 候选参数列表
- 调优报告

### 参考脚本
- 主要参考：`tune_noNKT_dims_resolution.R`
- 整合多个调优脚本的重复功能

---

## 阶段5：可视化 (`06_visualization.R`)

### 核心功能
1. **最终UMAP生成**：使用优化参数生成高质量UMAP
2. **多维度可视化**：按时间点、簇、细胞类型等多种方式可视化
3. **组合图生成**：创建组合展示图件
4. **多格式输出**：支持PNG、PDF、SVG等多种格式

### 输入
- 阶段4输出的Seurat对象和最佳参数
- 配置的可视化参数

### 输出
- 最终UMAP图件（多种格式）
- 簇组成趋势图
- 堆叠柱状图
- 组合展示图
- 可视化报告

### 参考脚本
- 整合：`generate_umap_nk_post.R`、`plot_umap_from_rds_noCluster6_timepoint_split.R`、`visualize_with_hidden_clusters.R`
- 统一风格和输出格式

---

## 主控制脚本 (`run_pipeline.R`)

### 核心功能
1. **流程协调**：按顺序执行各阶段脚本
2. **参数传递**：在各阶段间传递必要参数
3. **进度监控**：显示执行进度和状态
4. **错误处理**：完善的错误检查和恢复机制

### 输入
- 配置文件路径
- 可选的阶段覆盖参数

### 输出
- 完整的执行日志
- 各阶段的输出文件
- 最终状态报告
- 错误和警告汇总
